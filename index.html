Creating pages manually

If you prefer to not use the automatic generator, push a branch named gh-pages to your repository to create a page manually. Jekyll makes it easy to create site-wide headers and footers without having to copy them across every page. It also offers intelligent blog support and other advanced templating features.

How data was found
  • Google search with "'company' technology stack"
  • Engineering articles of company
  • Job Offers, especially for web developer
  • Quora, Stack Overflow
  • stackshare.io
  • Library Detector (chrome addin)
  • Analysis of web site sources
Notes
  • Most of the solutions, as a part of architecture they contain messaging and caching solutions
  • Things like JavaScript, HTML5, jQuery, CSS3, AJAX are assumed for every case (with good approximation)
  • CSS - most of the cases uses Sass or Less
  • Most cases used bower & grunt or alternatives
  • Underscore or lo-dash, those are another very popular competing libraries
